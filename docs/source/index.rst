Welcome to Lumache's documentation!
===================================

**Hackable-ESP-Device** Met deze instructies krijgt u een kopie van het project dat actief is op uw D1 Mini (of andere op ESP8266 gebaseerde boards) voor ontwikkelings- of hackdoeleinden.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
